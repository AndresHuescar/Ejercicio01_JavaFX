# Ejercicio 1 - JavaFX

## Descripción

Este proyecto corresponde al ejercicio 1 del bloque BC5 de JavaFX.

El objetivo del ejercicio es crear una aplicación básica de JavaFX que muestre una ventana con un botón que contiene el texto:

**"Haz clic aquí"**

## Tecnologías utilizadas

- Java
- JavaFX
- Maven
- IntelliJ IDEA
